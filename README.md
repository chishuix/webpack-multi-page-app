# webpack-multi-page-app

## 目录结构

```
|-- src/
|   |-- pages/
|       |-- common -- 公共文件
|           |-- index.js
|           |-- index.scss
|           |-- components/ -- 组件文件夹
|               |-- _header.html
|               |-- ...
|       |-- page-a
|           |-- index.html
|           |-- index.css
|           |-- index.js
|       |-- page-b
|           |-- index.html
|           |-- index.css
|           |-- index.js
|       |-- (更多页面)
```
